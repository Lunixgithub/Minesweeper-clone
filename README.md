# Minesweeper Projekt
Das Projekt ist ein Schulprojekt und wurde in BlueJ erstellt.

The Projekt was made for a school and was created in  Bluej.



## how to run the program
Lade das Programm herunter und öffne es in BlueJ. Erstelle ein neues GameManager Objekt und genieß das Zocken!

Download the repository and open it in BlueJ. Create a new GameManager object and have fun playing!
